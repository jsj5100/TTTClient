# JSJ_TTTClient
18100175 지능정보통신공학과 전성진

## 개요
1. 2022년 2학기 Network 수업의 결과입니다.
1. C# 으로 작성된 콘솔 코드입니다.
1. Soket 통신을 이용해 처음으로 작성하였습니다.

1. soket 통신중 클라이언트를 담당하는 코드입니다

1. 게임은 컴퓨터에 있는 cmd를 이용한 콘솔 게임입니다.
